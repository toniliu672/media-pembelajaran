Materi 1

Pengertian Web Development


Web development adalah istilah yang mengacu pada praktik membangun, membuat dan memelihara situs web1. Web development terdiri dari tiga divisi atau bagian, yaitu front-end development (client-side development), back-end development (client-side development), dan full-stack development2.

Tahapan dalam pengembangan website dapat berbeda-beda tergantung pada metodologi yang digunakan. Namun secara umum, tahapan dalam pengembangan website meliputi perencanaan (planning), perancangan (design), implementasi (coding), dan pasca-implementasi (post-deployment) dan pemeliharaan (maintenance).

Pada tahap perencanaan, tim pengembang memilih berbagai teknologi dan metodologi pengembangan software, mendefinisikan tujuan, dan memperkirakan durasi waktu dan resource untuk menyelesaikan proyek. Pada tahap perancangan, seluruh arsitektur informasi pada sebuah website dikembangkan dan prototipenya mulai dilakukan pengujian. Pada tahap implementasi, tim pengembang membagi setiap elemen website dan menggunakannya untuk membuat website fungsional. Tahap ini meliputi; pengembangan halaman depan setelah kerangka dasar (Shell) selesai dibuat. Shell website mengandung struktur navigasi dan tata letak situs tanpa konten web. Pada tahap pasca-implementasi dan pemeliharaan, tim pengembang melakukan pemeliharaan terhadap website yang telah dibuat.

Materi 2

Frontend Engineering

Front-end engineering adalah seorang insinyur yang berfokus pada tampilan dan nuansa situs beserta fungsinya. Pengembang front-end biasanya menjadi bagian dari tim yang mengembangkan, merenovasi, atau memelihara situs web, baik sebagai departemen internal dalam perusahaan atau sebagai bagian dari penyedia solusi web pihak ketiga.

Pekerjaan front-end engineering meliputi pengembangan antarmuka pengguna (user interface), pengembangan aplikasi web dan mobile, serta pengembangan game. Pengembang front-end biasanya menggunakan bahasa pemrograman seperti HTML, CSS, dan JavaScript untuk membuat tampilan website.

Backend Engineering

Backend adalah bagian dari website yang tidak terlihat oleh pengguna akhir. Backend terdiri dari server, aplikasi, dan database yang berfungsi untuk mengelola data dan memproses permintaan dari pengguna. Pengembang backend bertanggung jawab untuk membangun dan memelihara server, aplikasi, dan database yang mendukung website. Pengembang backend biasanya menggunakan bahasa pemrograman seperti PHP, Python, Ruby, Java, dan .NET untuk membuat aplikasi web.

Materi 3

Fullstack Developer

Full-stack development adalah pengembangan website yang melibatkan pengembangan front-end dan back-end. Pengembang full-stack bertanggung jawab untuk mengembangkan seluruh bagian dari website, termasuk tampilan dan fungsionalitasnya. Pengembang full-stack biasanya menggunakan bahasa pemrograman seperti HTML, CSS, JavaScript, PHP, Python, Ruby, Java, dan .NET untuk membuat website.

Materi 4

Others : UI UX Designer

Selain front-end, back-end, dan full-stack development, ada beberapa bagian pendukung lain dalam web development seperti UI/UX design, content writing, dan SEO. UI/UX design adalah proses merancang tampilan dan interaksi pengguna pada website. Tujuannya adalah untuk membuat website yang mudah digunakan dan menarik bagi pengguna.

Content writing adalah proses menulis konten untuk website. Konten harus informatif dan menarik bagi pengunjung website. SEO (Search Engine Optimization) adalah proses meningkatkan visibilitas website di mesin pencari seperti Google. Tujuannya adalah untuk meningkatkan jumlah pengunjung website melalui hasil pencarian organik di mesin pencari.
